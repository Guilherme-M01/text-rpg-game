# 🎮 Text-Based RPG Game

## 📖 About
This is a text-based RPG developed in Small Basic.  
The player makes choices that affect the progression of the story.

## ⚙️ Features
- Decision-making system  
- Interactive story  
- Basic game logic  

## 🛠️ Technologies
- Small Basic  

## 🚀 How to Run
1. Open the code in Small Basic  
2. Run the program  
3. Follow the instructions on screen  

## 📌 Future Improvements
- Combat system  
- Save/load system  
- More story branches
